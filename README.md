## ✨ Features

- **Standard Operations:** Addition, subtraction, multiplication, and division.
- **Advanced Math:** Includes Trigonometry (sin, cos, tan), Square Root (√), Power (^), and Percentages (%).
- **Full Keyboard Support:** Seamlessly use your Numpad or keyboard for fast calculations (Enter to equal, Backspace to delete, Escape to clear).
- **Safe Evaluation:** Built using ES6 Classes and strict logic instead of the vulnerable `eval()` method.
- **Premium UI:** Features dynamic hover effects, smooth transitions, and a responsive grid layout.

---

## 🛠️ Technologies Used

- **HTML5:** Semantic structure.
- **CSS3:** CSS Grid for layout, Custom Properties (Variables), and Keyframe Animations.
- **Vanilla JavaScript (ES6+):** Object-Oriented Programming (Classes) for clean and maintainable logic.
